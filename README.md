# leaflet-popupmovable
Make Leaflet's L.Popup movable and draw leadline.

[demo](http://)

### Usage
1. include the plugin file.
```
<script src="Leaflet.PopupMovable.js"></script>
```
2. Initialize plugin's class.
```
<script>
  new PopupMovable(your-L.Map);
</script>
```
